Ham Radio Extra Study Guide Assets

New here? Start with the official NCVEC question pool. This project turns that source into structured JSON, adds clear LLM-assisted summaries with rule-based validation checks, and uses the validated results to produce both the audio script and the PDF study workbook.

Study Workbook PDF Printable workbook derived from the same validated source data. Audio Book Rendered listening version for on-the-go study. Audio Script (Automation) Full narration text, primarily useful for downstream automation. Prose Question Pool JSON (Automation) Structured validated data for programmatic processing and integrations.
